# Design of 8 bit Parity generator using sky130nm technology<br/>
---
*In this project 8-bit odd and even parity generator (dvsdparitygen_8bit) is implemented using google skywater130 PDK (Process Design Kit). This project give a detailed view in generating a final layout to print photomasks used in the fabrication of a behavioral RTL (Register-Transfer Level) of an 8-bit parity generator, using SkyWater 130 nm PDK*<br/>
